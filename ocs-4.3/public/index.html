About

This is an online ruleset of the OCS 4.3 Series Rules as provided by the OCS Depot. Rule references have been hyperlinked. These rules are the intellectual property of the OCS designers who hold all appropriate copyright.


My official online ruleset resides at https://dornshuld.chemistry.msstate.edu/rules/ocs-4.3/. This skeleton site was created by request for the purpose of having a slimmed down version for offline use.


The source code for this ruleset can be found on Github. This site has been created using Wowchemy, Hugo, and Blogdown. Please contact me (edornshuld{at}chemistry{dot}msstate{dot}edu) if you find any errors.
